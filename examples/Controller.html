Controller ​

装饰器 Controller 意在收集目标注解对象内部注册的路由，是 sanic 框架中 blueprint 蓝图系统的自动化实现。

  • 无参控制器
  • 有参控制器

无参控制器 ​

使用无参控制器，控制器 uri 默认以全小写的类名为值。

python
@Controller
class PersonController:
	pass
